Projetos de Destaque

Calculadora de IMC em Python com Histórico (CSV)


Calculadora de IMC simples em Python que implementa a fórmula padrão da OM S. Salva o histórico de resultados em um arquivo JSON para persistência de dados.

Gerador de Senhas Seguras em Python


Ferramenta simples em Python para gerar senhas aleatórias e de alta entropia. O script permite configurar o tamanho da senha e quais tipos de caracteres (símbolos, números, letras) devem ser incluídos, garantindo máxima segurança.

Ferramenta de consulta rápida desenvolvida em Python 3.x para interagir com o endpoint /anime da Jikan API. O foco é na comunicação cliente-servidor, validação de resposta HTTP e exibição organizada de informações extraídas de JSON complexo diretamente no terminal.
